Tailwind setup

In Terminal-

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p


In tailwind.config.js file-

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}


In index.css file-

@tailwind base;
@tailwind components;
@tailwind utilities;



Now you are ready to go
